### 2026-02-18
**アクセシビリティ**  
**関数式、アロー関数**  
**組み込み高階関数**

#### ✏️ 学んだこと、実行したこと
- アクセシビリティ
  - Perceivable：知覚可能
    利用者の感覚全てに対して、知覚できない状態を避ける。
  - Operable：操作性
    Webサイト等のUIにおいて、全てのユーザーが機能を簡単に使いこなせる状態
  - Understandable：理解可能
    誰もが、情報を迷わず読み解ける状態のこと
  - Robust：堅牢性
    どんな環境でも、いつでも、ブラウザが正しく表示できるようにする
- 関数式・アロー関数
  - 生成
  - 代入
  - 演算
  - 受け渡し（関数の引数に渡したり、戻り値として受け取ったり）
- JavaScriptにおける高階関数
  - Array.prototype.map→ map関数は、配列の各要素に関数を適用し、その結果からなる新しい配列を作成します。
  - Array.prototype.filter→ filter関数は、特定の条件に一致する要素だけを含む新しい配列を作成します。
  - Array.prototype.reduce→ reduce関数は、配列の各要素を順に処理し、単一の結果値に統合します。
